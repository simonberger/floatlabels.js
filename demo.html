floatlabels.js

Introduction to floatlabels

The Float Label Pattern found his success after Brad Frost wrote a blog post about Float Label Patterns. The idea is based on a Dribbble Shot by Matt D Smith. The Pattern is easy to explain. On User Interaction with an input field the placeholder value moves up, and is displayed above the typed text. This Plugin offers you the possibility to upgrade your forms with this famous pattern. This project is a fork of floatlabels.js bye Marcus Pohorely - clubdesign

Demo

Installation & Dependencies

JS-Part

Include the latest release of jQuery. The Google AJAX Libraries API can be used for this but you could also host the library yourself.
Download floatlabels.min.js, upload the file to your server & include it in your code AFTER jQuery.

Currently floatlabels.js is dependant from jQuery 1.8+. Lower Versions have not been tested.

CSS-Part

You need to include some key css classes into your project. They are defined in demo.css. You can change them to your own need. Details are described under usage.

Usage

  1. Include jquery.min.js and floatlabels.min.js
  2. initialized floatlabel with simple call:

    <script> $('input').floatlabel(); </script>
  3. Add plugin options if the default functionality should be changed

    slideInput Default: true
    If true the input field will slide down (height) to fit the floating label.

    typeMatches Default: /text|password|email|number|search|url/
    This Regex Expression is triggered at the initializiation of the script. It checks the type="text" if the element the plugin is triggered on is valid. Some advanced users might want to expand this to the new HTML5 textfield types.

  4. Include the necessary styles of demo.css to your project and change them on your needs.
    Here is a description of the used classes:

    floatlabel-label

    It's the floating label itself which holds the left position and style of the displayed placeholder text

    floatlabel-label-inactive, floatlabel-active

    Just adds display: none, display: inline-block and the top position of the label to get some fancy animation

    floatlabel-input

    Initializing the plugin adds this class to all input fields.
    If you use slideInput: false (default true) add the needed padding either here or in a general project class.

    Note: You might need to use !important marker to overwrite your own padding values of input fields

    floatlabel-input-slide

    input field Padding added with option slideInput: true (default true)

    floatlabel-label-focus

    Added to the label if the input field gets focus. Gives teh Possibility to overwrite color or other styles of the placeholder text

  5. The script cycles through all your selected input fields and converts them to inputs with floated labels. Done.
